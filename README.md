#Iphone 17

## Projeto de Estudo: React e Tailwind

Este é um projeto de estudo que utiliza React e Tailwind CSS. O objetivo é aprender e aplicar conceitos de desenvolvimento front-end utilizando essas tecnologias modernas. Este projeto foi desenvolvido como parte dos estudos propostos pelo canal **DevClub**.

## Tecnologias Utilizadas

React: Biblioteca JavaScript para construção de interfaces de usuário.  
Tailwind CSS: Framework CSS utilitário para estilização rápida e eficiente.
